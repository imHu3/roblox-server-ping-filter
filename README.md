# Roblox Server Ping Filter

It simply finds the best servers.

## Features
**Sort by Ping**: Find servers with the lowest latency instantly.
**Direct Join**: One-click hop into the right server.

## Installation
1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select the folder containing this extension (`Ping`).
5. Go to a Roblox game page and find the fastest server!

## How it works
This extension bypasses Roblox's CSP limits by making background requests and safely appending server information right into the server list page.
